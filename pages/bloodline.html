Projects

Geo-Location Database

This Project is about create a dynamic database of places with information about known places in history. This tool will later be integrated with the Bloodline Family tree to help users easily add place location. The database will also contain other information about the location such as founding date, surrounding counties and such.

Bloodline Family Tree

This project will allow researchers to create an account and build their family tree. They can invite users to collaborate with them and build a tree together.
